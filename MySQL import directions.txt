Prerequisites:
1. XAMPP 
2. File to be imported (*.sql)

Steps: 
1: Ope up XAMPP and MySQL module, and go to localhost/phpMyAdmin.
2: Create a new database name, with the same name as the database to be imported for easier configuration.
3: Leave it blank and find the Import button at the upper portion of the current website. Click it.
4: Under the "File to import" field, click [Browse...] and locate the file to be imported on the pop up window.
5: Go all the way to the bottom of the current website and click [Import].
